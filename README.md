# AplicacionFlask
# 1. Ir al repositorio en GitHub, hacer clic en Code → Download ZIP y descomprimir el archivo en una carpeta.
# 2. Abrir la carpeta del proyecto y luego abrir una terminal dentro de esa carpeta (Shift + clic derecho → “Abrir ventana de comandos aquí” o usar la terminal de VS Code).
# 3. Crear el entorno virtual:python -m venv venv
# 4. Activar el entorno virtual: venv\Scripts\activate
# 5. Instalar las dependencias (si existe el archivo requirements.txt): pip install -r requirements.txt
# 6. Ejecutar el archivo app.py: python app.py
# 7. Abrir el archivo index.html: hacer doble clic sobre index.html o abrirlo con Live Server desde VS Code.
# 8. Mantener la terminal abierta mientras se usa el proyecto para que app.py siga ejecutándose.
